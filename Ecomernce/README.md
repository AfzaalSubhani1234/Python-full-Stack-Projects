# 🛒 Full Stack E-Commerce Platform

A modern full-stack e-commerce application built using **Django REST Framework (backend)** and **React JS (frontend)** with JWT authentication and a production-ready architecture.

---

## 🚀 Tech Stack

### Backend
- Django
- Django REST Framework
- Simple JWT Authentication
- PostgreSQL (Production)
- CORS Headers

### Frontend
- React JS
- Axios
- React Router DOM

### Dev Tools
- Git & GitHub
- Postman (API testing)
- Environment Variables (.env)

---

## ✨ Features

### 👤 User Features
- User registration & login (JWT)
- Secure authentication system
- View products
- Add to cart
- Place orders
- View order history

### 🛠 Admin Features (optional future)
- Add/update/delete products
- Manage orders
- Manage users
